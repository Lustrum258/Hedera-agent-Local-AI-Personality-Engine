## 安全注意事项

- **API Key**：请勿将 API Key 提交到代码仓库。使用环境变量或 `config.yaml`（已在 `.gitignore` 中忽略）
- **密码**：默认密码 `hedera2024` 仅用于开发环境，生产环境请修改
- **网络暴露**：默认监听 `0.0.0.0:36313`，如仅本地使用请改为 `127.0.0.1`
