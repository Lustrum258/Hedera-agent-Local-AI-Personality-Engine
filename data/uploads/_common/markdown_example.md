## 文本样式

- **粗体** 用 `**` 包裹
- *斜体* 用 `*` 包裹
- ~~删除线~~ 用 `~~` 包裹
- `行内代码` 用反引号包裹
- 也可以混合使用 **粗体 *和斜体***

## 代码块

```python
def hello(name):
    print(f"你好, {name}!")

```

```javascript

    console.log(`Hello, ${name}!`);
```

## HTML 混用

Markdown 里可以直接写 HTML：

这里是被折叠的内容，支持 Markdown **语法**。

</details>
