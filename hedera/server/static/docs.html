概述

Hedera 是一个轻量级 AI Agent 框架，基于 Python 标准库构建，零外部依赖（HTTP 服务使用内置 http.server）。核心设计理念：人格驱动。每个会话绑定一个灵魂文件（SOUL.md），决定了 AI 的说话风格、价值观和行为准则。

特性

  • 人格切换 — 通过 SOUL.md 定义人格，创建会话时选择，终生绑定
  • 独立会话 — 每个会话有自己的历史、人格、记忆
  • 跨会话记忆 — 自动注入其他会话的关键上下文和经验准则
  • 自省系统 — 定期复盘对话，提炼经验准则
  • 噪声层 + 滑块光谱 — 动态调节性格状态
  • 图像生成 — 支持 OpenAI 兼容 API
  • 实时工具进度 — 前端实时显示工具调用链
  • 自提问机制 — 模拟好奇心，3 次无回答自动闭嘴
  • WebUI — 毛玻璃风格界面，自定义下拉组件，动画效果，支持 9 种语言
  • 零依赖 HTTP 服务 — 使用 Python 内置 http.server
  • 插件系统 — 可扩展的工具架构

聊天流式响应

POST /chat 返回 Content-Type: application/x-ndjson，每行一个事件：

{"type":"tool","name":"web_search","args":{"query":"..."},"status":"running"}
{"type":"tool","name":"web_search","args":{"query":"..."},"status":"success"}
{"type":"result","response":"最终回答","session_id":"_default","files":[]}

响应头 X-Request-Id 可用于并行轮询 GET /chat/progress?req={id}。

图像生成

配置 image_gen 节后，AI 自动调用 generate_image 工具。endpoint 为空时自动推导：

  • https://xxx.com/v1/chat/completions → https://xxx.com/v1/images/generations
  • https://cdn.xxx.cn → https://cdn.xxx.cn/v1/images/generations

可在设置面板直接填写。

自省系统

  • 每 5 分钟检查最近对话
  • 4 维度复盘：学到了什么、改进点、原则提炼、盲区修正
  • 置信度 ≥ 4 的反思蒸馏为经验准则
  • 自提问：从对话历史抽取关键词生成问题，冷却 15-40 分钟
  • 连续 3 次提问无人回复 → 自动闭嘴，用户发消息后恢复

# 初始化工作目录
python -m hedera init
